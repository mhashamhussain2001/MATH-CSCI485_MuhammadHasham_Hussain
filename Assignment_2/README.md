# Assignment 2 - Recursive Feature Elimination

This folder contains:
1) Colab implementation (Source Code IPYNB)
2) Report PDF

This project uses the **scikit-learn Diabetes dataset** to:
1) explore the dataset (summary stats + plots),
2) build a **baseline Linear Regression** model,
3) apply **Recursive Feature Elimination (RFE)** to evaluate performance as features are removed,
4) report the most important features and the final selected subset,
5) include a short reflection on feature selection.
